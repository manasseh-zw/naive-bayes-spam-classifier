# Naive Bayes Spam Classifier (Group 3)

This repository contains a single, presentation-ready notebook for an AI Modeling course practical.

## Model Chosen

- Naive Bayes Classification

## Project Objective

Build a machine learning classifier that predicts whether a message is:

- Spam (1)
- Not Spam / Ham (0)

## Notebook

- Group 3 - Naive Bayes Spam Classification (Colab Ready).ipynb

## Assignment Coverage (Step 1 to Step 8)

The notebook is structured to directly answer the assignment requirements:

1. Clarify the task, success metrics, and constraints
2. Data collection with source explanation
3. Exploratory Data Analysis (EDA)
4. Data cleaning and preprocessing
5. Train/validation/test split
6. Model selection (Naive Bayes variants)
7. Model training with hyperparameter tuning
8. Model evaluation using relevant metrics

## What Is Included

- End-to-end text classification pipeline
- Modern Python ML stack (scikit-learn, pandas, numpy, seaborn, matplotlib)
- Naive Bayes theory section with equations
- Evaluation metrics: Accuracy, Precision, Recall, F1, ROC-AUC
- Confusion matrix and ROC curve
- Live inference demo (type a message, get spam/ham prediction)
- Token-level contribution explanation for model interpretability

## Dataset

The notebook attempts to load an SMS spam dataset from Hugging Face first, with fallback handling for robust execution in Colab.

## How To Run (Google Colab)

1. Open Google Colab
2. Upload the notebook file from this repo
3. Run all cells from top to bottom
4. Use the live inference section to test custom messages
